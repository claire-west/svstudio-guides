1.1

What are dictionaries?

Dictionaries are used to determine which phonemes should be used to represent a word. For example, entering "hello" as a lyric will cause the phonemes hh ax l ow to be used when rendering the audio output.

For more information about lyrics, phonemes, and dictionaries, see the full guide on Entering lyrics & phonemes for better pronunciation & timing, and how to use dictionaries.

Dictionaries are sometimes used to sing in unsupported languages using the phonemes available from the supported ones. For example, Solaria comes with a "Japanese to Spanish" dictionary which is meant to be used to produce Spanish-sounding output using the Japanese phonemes made available via cross-lingual synthesis. In these cases, this language will be denoted as the "Target Language" below.

1.2

Using Dictionaries

To use a dictionary, download the .json file and put it in the appropriate dictionary folder for your voice database's native language, which can be located with the "Open Scripts Folder" option within SynthV Studio Pro.

Dictionaries are always loaded from the folder that matches the selected voice database's native language.

This means even if you are using Solaria (a native English vocal) to sing in Japanese, SynthV Studio will load dictionaries from the english-arpabet folder. While this is somewhat counter-intuitive, this means that you should copy any dictionaries you want to use with Solaria into the english-arpabet folder, even if they are not English dictionaries.

Selecting a different language for cross-lingual synthesis does not change which dictionaries are loaded for the current voice database.
